\# 📊 Unemployment Analysis in India - Internship Project



\## 📌 Project Overview

This project analyzes unemployment trends in India using Python. It includes data cleaning, visualization, and COVID-19 impact analysis to understand how unemployment changed over time.



\---



\## 🛠️ Tools Used

\- Python

\- Pandas

\- Matplotlib

\- Seaborn

\- VS Code



\---



\## 📁 Dataset Used

\- unemployment in india.csv

\- unemployment\_rate\_upto\_11\_2020.csv



\---



\## 🔍 Steps Performed

1\. Data loading

2\. Data cleaning

3\. Date conversion

4\. Trend analysis

5\. Monthly trend analysis

6\. COVID-19 impact analysis

7\. Correlation analysis
